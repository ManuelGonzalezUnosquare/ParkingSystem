# Technology Stack

## Backend

- **Framework:** NestJS (Node.js)
- **Language:** TypeScript
- **ORM:** TypeORM
- **Database:** MySQL 8.0
- **Validation:** Class-validator & Class-transformer

## Frontend

- **Framework:** Angular
- **State Management:** RxJS

## Tooling & DevOps

- **Monorepo Management:** Nx
- **Automation:** Husky (Git Hooks)
- **Documentation:** Mermaid.js (for ERD)
- **CI/CD:** GitHub Actions
